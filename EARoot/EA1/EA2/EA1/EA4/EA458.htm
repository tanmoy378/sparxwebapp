This table is used to calculated Average Degradation Rate for a given Element, Feature, Asset Remaining Lift years band combination. This is then used in datamart reporting views to calculate the predicted year of expiry for an asset based on average degradation of similar (similar meaning - same element and feature values) operation property asserts across Network Rail. <br/>
