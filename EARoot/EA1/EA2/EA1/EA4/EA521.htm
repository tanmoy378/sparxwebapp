A formal record of a work undertaken that records the time, location and scope ( against a single Asset, Asset System or Component) of the work. It provides audit trail and data required for compliance reporting.<br/><br/>Previously called Inspection Record.<br/>This can be attributed using Work Record Attribute.<br/><br/>CIM Entity : work Record<br/>
