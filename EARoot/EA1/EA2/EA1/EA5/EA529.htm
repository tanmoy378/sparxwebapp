Stores list of all the tables in the data mart. <br/><br/>CIM- DESCRIPTION<br/>A set of data persisted within a system, or input/output to/from a system.<br/><br/>For persistent data, depending on the technology used the set can manifest as:<br/><ul>
<li>a table in a relational database,</li><li>a set of labeled nodes within a graph database or</li><li>a file contained within a filing system.</li><li>a message containing message elements</li><li>a file containing fields</li></ul>
<br/>For transitory data, the set can manifest as:<br/><ul>
<li>data collected/transmitted via an interface</li><li>a screen or dashboard containing fields</li><li>a report containing fields</li><li>a data event containing data event elements</li></ul>
<br/>This could also represent a script.<br/><br/>
