In this context, this refers to assessment compliance, as defined in CIV035MOD01. The standard defines a number of business rules that mandate which assets are subject to the structural assessment control, when and how often they need to be assessed. This entity will track the compliance position of each asset, which is a key asset management metric  for example, compliance metrics inform the creation of the task list.<br/>
